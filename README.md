## 🎯 ریون دانلودر چیست؟

**Raven Downloader** یک راهکار ساده و رایگان برای دانلود فایل‌های اینترنتی از طریق گیت‌هاب است. 

از آنجایی که گیت‌هاب در ایران بدون فیلتر است، شما می‌توانید:
1. فقط **یکبار** با فیلترشکن لینک مستقیم فایل را پیدا کنید
2. بقیه کار (دانلود، ذخیره‌سازی و دریافت فایل) **بدون فیلترشکن** انجام می‌شود

### ✨ ویژگی‌ها

- 🆓 کاملاً **رایگان**
- 🚫 **بدون نیاز به فیلترشکن** برای دانلود
- 🔒 **بدون نیاز به VPN** (به جز یکبار برای یافتن لینک)
- 📦 پشتیبانی از فایل‌های **حجیم** (تا چند گیگابایت)
- 🔄 **تکه‌تکه کردن خودکار** فایل‌های بزرگ
- 📱 قابلیت دانلود از طریق **موبایل و کامپیوتر**
- 🏷️ **نام‌گذاری دلخواه** فایل‌ها
- 🌐 **مخزن عمومی یا خصوصی** (به انتخاب شما)

## 🎬 آموزش تصویری

1. **Fork کنید**: روی دکمه Fork در بالای صفحه کلیک کنید
2. **لینک را پیدا کنید**: با فیلترشکن، روی فایل کلیک راست → Copy link address
3. **اجرا کنید**: به تب Actions بروید → Run workflow → لینک را Paste کنید
4. **دانلود کنید**: فایل‌های `Raven_*` را از مخزن خود دانلود کنید
5. **استخراج کنید**: با نرم‌افزار 7-Zip همه تکه‌ها را باز کنید

| **Linux** | p7zip | `sudo apt install p7zip-full` |
| **macOS** | Keka | [keka.io](https://keka.io) |

## ❓ سوالات متداول

<details>
<summary><b>آیا گیت‌هاب در ایران فیلتر است؟</b></summary>
خیر. گیت‌هاب معمولاً بدون فیلتر در دسترس است.
</details>

<details>
<summary><b>محدودیت حجم چقدر است؟</b></summary>
مخزن رایگان ۱GB فضا دارد. فایل‌ها به صورت تکه‌های ۹۵MB ذخیره می‌شوند.
</details>

<details>
<summary><b>چرا فایل‌ها تکه‌تکه می‌شوند؟</b></summary>
گیت‌هاب محدودیت ۱۰۰MB برای هر فایل دارد. تکه‌تکه کردن این محدودیت را دور می‌زند.
</details>

<details>
<summary><b>آیا فایل‌های من خصوصی هستند؟</b></summary>
مخزن Fork شده شما عمومی است. می‌توانید در Settings آن را **Private** کنید.
</details>
